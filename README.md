# Знайка 🐰

**Знайка** — Telegram-бот с ИИ-репетитором по математике для школьников. Бот помогает детям понять математику через дружелюбное общение с персонажем — зайцем Знайкой.

## Возможности

- **Обучение с ИИ** — общение с ИИ-репетитором через текст, голосовые сообщения и изображения
- **Проверка решений** — отправка фото домашних заданий для проверки
- **Подписка** — система оплаты через Telegram Payments
- **Обратная связь** — отдельный бот для сбора отзывов пользователей
- **Согласие на обработку данных** — юридически корректный сбор согласий с защищённым хранением

## Требования

- Python 3.10+
- Внешний AI-сервис с webhook-интерфейсом

## Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/your-repo/znaika.git
   cd znaika
   ```

2. **Установите зависимости:**
   ```bash
   pip install aiogram aiohttp aiosqlite python-dotenv
   ```

3. **Настройте переменные окружения:**
   ```bash
   cp .env-example .env
   ```

   Заполните `.env` файл:
   ```env
   BOT_TOKEN_MAIN=ваш_токен_основного_бота
   BOT_TOKEN_FEEDBACK=ваш_токен_бота_обратной_связи
   PAYMENT_TOKEN=токен_платёжного_провайдера
   AI_WEBHOOK_URL=https://ваш-сервер/ai-endpoint
   AI_WEBHOOK_URL_CHECKING_HOME_WORK=https://ваш-сервер/homework-ai-endpoint
   ```

4. **Настройте бот обратной связи:**

   В файле `feedbackbot.py` замените ID администраторов:
   ```python
   ADMINS = [
       123456789,  # Ваш Telegram ID
   ]
   ```

## Запуск

**Основной бот:**
```bash
python main.py
```

**Бот обратной связи:**
```bash
python feedbackbot.py
```

## База данных

Бот использует SQLite для хранения согласий пользователей с защитой через цепочку хэшей (blockchain-подобная структура):

- `user_id` — Telegram ID пользователя
- `username` — имя пользователя
- `doc_version` — версия документа согласия
- `timestamp` — время согласия (UTC)
- `prev_hash` / `curr_hash` — SHA-256 хэши для защиты целостности

## Безопасность

- Согласия защищены криптографической цепочкой хэшей
- Все данные хранятся локально в SQLite
- Поддержка HTTPS для webhook-запросов
